Code Runner
How To Play:
- Through the process of elimination, you have to crack the code from the hints provided on the left.
- If you guess incorrectly, you will lose half a heart. This will continue till you lose all your hearts, signalling game over.
- Each level is a randomised code, and the highscore is your highest level achieved.
